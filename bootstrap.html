HTML
Özetle;

HTML bir web sitesinin iskeleti olarak düşünülebilir.Hiper Metin İşaretleme Dili web sayfalarını oluşturmak için kullanılan standart metin işaretleme dilidir.Dilin son sürümü HTML5'tir. Bir programlama dili olarak tanımlanamaz.

CSS
Özetle;

CSS,HTML'e ek olarak metin ve format biçimlendirme alanında fazladan olanaklar sunan bir işaretleme dilidir.HTML'in yapısal içeriğini sunum katmanından ayırarak web sayfalarının daha esnek ve tuttarlı bir şekilde stilize edilmesini sağlar.

JAVASCRİPT
Özetle;

Javascript geliştiricilerin etkileşimli web sayfaları oluşturmak için kullandığı programlama dilidir.Javascript işlevleri sosyal medya akışlarını yenilemekten animasyonlar ve etkileşimli haritalar göstermeye kadar bir web kullanıcısının deneyimini iyileştirebilir.

Bootstrap bir CSS kütüphanesidir.Web uygulamalarının hızlı,kullanışlı ve daha duyarlı hale getirilmnesini kolaylaştırır.

HTML,CSS ve Javascript'e dayalıdır.Bu nedenle bu teknolojilere aşinaysanız Bootstrap'ı kolayca öğrenebilirsiniz.
